# 网格布局

网格布局就是把一个 div 分成 N 个部分（N = 12,16,24...），每个部分无空隙或者有空隙。

# 引入 Grid 组件

import { Col, Row } from "vue-wheel-demo";

export default {
  components: {
    "g-col": Col,
    "g-row": Row
  }
};

注意

使用该组件之前，请在 css 中开启 border-box

* {box-sizing: border-box;}

# 设置 gutter
