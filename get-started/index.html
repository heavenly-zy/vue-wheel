# 添加 CSS 样式

使用本框架前，请在 CSS 中开启 border-box

*，*::before,*::after{ box-sizing: border-box }

IE 8 及以上浏览器支持此样式。

# 引入 vue-wheel-demo

示例：

  1. 引入 Button 组件，并注册
import Vue from 'vue'
import { Button } from 'vue-wheel-demo'
import 'vue-wheel-demo/dist/index.css'

export default {
  name: "app",
  components: {
    HelloWorld,
    "g-button": Button
  }
};
  1. 使用 <g-button> 标签在页面中显示 Button 组件
<g-button>按钮示例</g-button>
